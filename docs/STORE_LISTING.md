**NoteFlow**

## 简短描述（132 字符以内）

一键导入哔哩哔哩、YouTube、播客、网页、AI 对话到 NotebookLM。批量选择，后台写入，优雅高效。

---

## 详细描述（纯文本版）

NotebookLM 是最好的 AI 知识工具，但把内容导入进去太麻烦了。NoteFlow 帮你解决这个问题。


🎬 **视频字幕导入**

支持导入哔哩哔哩视频字幕与弹幕、YouTube 视频内容。在视频页面打开扩展，自动识别当前视频，一键导入。NotebookLM 自动解析字幕生成笔记。

支持 youtube.com/watch、/playlist、/@频道 等格式，可限制导入最近 N 个视频，勾选批量导入。


🎙️ **播客内容导入**

粘贴小宇宙播客链接，自动获取节目信息和单集列表，一键导入到 NotebookLM。


🌐 **网页内容导入**

打开扩展的「网页」标签，自动列出当前浏览器所有窗口的标签页。按窗口分组清晰展示，支持全选/取消全选，勾选目标网站后一键导入到 NotebookLM。无需手动复制粘贴 URL。


💬 **AI 对话导入**

在 Claude、ChatGPT 或 Gemini 的对话页面打开扩展，自动提取问答对，按需勾选导入，格式化为结构化内容存入笔记本。


⚡ **后台批量导入**

选中多条内容后点击底部全局导入按钮，后台自动逐条写入 NotebookLM。并行发送请求，进度实时显示（1/5 → 2/5 → ... → 5/5），无需打开新标签页。


🔑 **多账号支持**

支持 Google 账号登录管理，自动检测当前 NotebookLM 激活账号。自由切换目标笔记本，导入内容自动发送到选中笔记本。


📋 **导入历史记录**

自动记录最近 100 条导入记录，随时回溯。


🌐 **中英双语界面**

自动适配浏览器语言，也可手动切换。


🔒 **安全与隐私**

- 完全免费，无需注册登录
- 纯客户端运行，不上传任何数据到第三方服务器
- 开源代码，可审计
- 符合 Chrome Manifest V3 最新安全规范

## 英文版（English Version）

### Extension Name

**NoteFlow**

### Short Description (132 chars)

Import Bilibili, YouTube, podcasts, web pages & AI chats into NotebookLM. Batch select, background import, elegant and efficient.

### Detailed Description

NotebookLM is the best AI knowledge tool, but getting content into it is a pain. NoteFlow solves this.

🎬 **Video Subtitle Import**

Import Bilibili video subtitles and YouTube content. Open the extension on a video page — auto-detect and one-click import. NotebookLM auto-parses transcripts for notes.

Supports youtube.com/watch, /playlist, /@channel. Select videos to import, limit to latest N.

🎙️ **Podcast Import**

Paste Xiaoyuzhou podcast links. Auto-fetch episode list, one-click import to NotebookLM.

🌐 **Web Page Import**

Open the "Web" tab in the extension. See all browser tabs grouped by window. Select/deselect all, check individual tabs, one-click import to NotebookLM. No manual URL copying.

💬 **AI Conversation Import**

Open the extension on Claude, ChatGPT, or Gemini conversation pages. Auto-extract Q&A pairs, selectively import valuable exchanges.

⚡ **Background Batch Import**

Select multiple items, click the global import button. Background parallel processing with real-time progress (1/5 → 2/5 → ... → 5/5). No new tabs needed.

🔑 **Multi-Account Support**

Google account login management. Auto-detect active NotebookLM account. Switch notebooks freely.

📋 **Import History**

Last 100 imports automatically recorded.

🌐 **Bilingual UI**

Auto-detects browser language with manual toggle.

🔒 **Privacy & Security**

- 100% free, no account required
- Runs entirely in your browser — zero data sent to third-party servers
- Open source
- Chrome Manifest V3 compliant
