# 更新日志: v1.0.0 — 初始创建

## 变更概要

首次创建 Skill 文档，定义"目标导向型项目复盘与笔记自动生成助手"的核心能力与规范。

## 变更内容

### ✨ 新增
- **技能定义:** 明确 AI 角色的核心职责为"将零散的开发进展自动沉淀为结构化的复盘笔记"
- **触发机制:** 定义三类触发条件（明确宣告、状态转变、阶段性交付）
- **分析流程:** 建立"追溯过去 → 提炼本质 → 识别代价"三阶段分析管道
- **输出模板:** 设计标准化的目标归档模板，含背景、关键转折点、经验沉淀三大区块
- **执行约束:** 规定严禁废话、自动迭代、防重复归档等行为边界
