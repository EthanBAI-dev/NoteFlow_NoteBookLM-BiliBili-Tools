## 版本号规范

- **MAJOR**: 模板结构重大变更，不向后兼容
- **MINOR**: 新增 section 或字段，向下兼容
- **PATCH**: 文案修正、格式微调、错别字修复

## 文件命名规范

```
YYYY-MM-DD-v{版本号}-{简述}.md
```

示例: `2026-06-08-v1.1.0-优化skill文档结构.md`
