## 导入你的第一个网页

<Steps>
  <Step title="打开 NotebookLM 笔记本">
    在浏览器中打开 [NotebookLM](https://notebooklm.google.com)，创建或进入一个笔记本。

    <Warning>
      请确保打开的是**笔记本内部页面**（不是首页），否则导入会失败。
    </Warning>
  </Step>
  <Step title="打开 NoteFlow 侧面板">
    打开 Chrome 右侧的 NoteFlow 侧面板。扩展会自动检测当前标签页的类型，匹配对应的导入工具。
  </Step>
  <Step title="选择 Notebook 并导入">
    1. 在侧面板中选择目标 Notebook（或保持默认）
    2. 当前导入工具会自动显示
    3. 点击「导入到 NotebookLM」按钮
    4. 等待几秒，内容会自动出现在 NotebookLM 的来源列表中
  </Step>
</Steps>

<Tip>
  **更快的方式**：在任意网页上右键 →「导入到 NotebookLM」，无需打开侧面板。
</Tip>
