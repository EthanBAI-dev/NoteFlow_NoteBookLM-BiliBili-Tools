title: 'YouTube 导入'
description: '导入 YouTube 视频、播放列表、频道到 NotebookLM'
---

## 功能

在浏览器中打开 YouTube 页面后，NoteFlow 侧面板自动检测并显示导入界面。

<Steps>
  <Step title="打开 YouTube 页面">
    打开一个 YouTube 视频、播放列表或频道页面。
  </Step>
  <Step title="自动查询">
    侧面板自动识别当前 YouTube 页面并获取视频列表。也可手动粘贴链接后查询。
  </Step>
  <Step title="选择视频">
    勾选要导入的视频。播放列表/频道支持"Load more"分页加载更多视频。
  </Step>
  <Step title="导入到 NotebookLM">
    点击导入，扩展将视频 URL 批量发送给 NotebookLM，由 NotebookLM 原生解析视频内容。
  </Step>
</Steps>

## 支持的页面类型

| 类型 | 说明 |
|---|---|
| 单个视频 | `youtube.com/watch?v=xxx` 或 `youtu.be/xxx` |
| 播放列表 | `youtube.com/playlist?list=xxx` |
| 频道 | `youtube.com/@xxx` 或 `youtube.com/channel/xxx` |

## 说明

- NotebookLM 原生支持 YouTube 视频解析，扩展只需发送视频 URL 即可
- 播放列表/频道批量导入进度实时显示
- 支持 SPA 页面导航检测（YouTube 单页应用路由变化自动触发刷新）
