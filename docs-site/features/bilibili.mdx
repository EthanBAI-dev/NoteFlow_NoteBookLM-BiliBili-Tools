## 功能

在浏览器中打开 Bilibili 视频页面、UP 主空间或收藏夹后，NoteFlow 侧面板会自动检测并显示字幕提取界面。

<Steps>
  <Step title="打开 Bilibili 页面">
    打开一个 Bilibili 视频、UP 主空间或收藏夹页面。
  </Step>
  <Step title="自动识别">
    侧面板自动识别 Bilibili 页面，显示视频信息列表和字幕状态。
  </Step>
  <Step title="选择视频（可选）">
    - 单个视频：自动选中当前视频
    - 多个视频：勾选需要的视频，支持全选/取消全选
    - 支持"加载更多"分页（每页 100 个）
  </Step>
  <Step title="导入到 NotebookLM">
    点击「导入到 NotebookLM」，选中的视频字幕会自动出现在 NotebookLM 的来源列表中。
  </Step>
</Steps>

## 支持的页面类型

| 类型 | 说明 |
|---|---|
| 单个视频 | `bilibili.com/video/BVxxx` — 自动识别，支持多 P（分P） |
| UP 主空间 | `bilibili.com/space/xxx` — 列表所有投稿视频 |
| 收藏夹 | `bilibili.com/list/xxx` — 分析收藏夹所有视频 |
| 系列/合集 | 自动识别视频所属系列，展示全部章节 |

## 输出格式

| 格式 | 说明 |
|---|---|
| `.txt` | 纯文本，带时间戳 |
| `.md` | Markdown 格式 |
| `.json` | 结构化 JSON（含时间戳、文本、BVID） |
| `.srt` | SRT 字幕格式 |

## 导出模式

| 模式 | 说明 |
|---|---|
| **下载单个** | 点击每个视频旁的 TXT 按钮，单独下载当前视频字幕 |
| **分开下载** | 选中多个视频，打包下载为 ZIP（每个视频独立文件） |
| **合并下载** | 选中多个视频，合并为一个文件下载（适用于系列教程） |

## 字幕状态

扩展会自动检测视频是否有 CC 字幕。无字幕的视频会显示提示，无法导入。

<Note>
  合并模式最适合系列教程 — 将多个视频字幕合并为一个文档导入，仅占 1 个 NotebookLM 来源配额。
</Note>
