---

## 功能

NoteFlow 自动检测当前标签页类型。如果是普通网页，则显示网页导入界面。

<Steps>
  <Step title="打开 NoteFlow 侧面板">
    打开 NoteFlow 侧面板，扩展自动识别当前标签页。
  </Step>
  <Step title="查看页面信息">
    侧面板顶部显示当前页面的标题和 URL，确认是你想要导入的页面。
  </Step>
  <Step title="选择标签页">
    - **当前标签页**：默认选中，直接导入
    - **所有标签页**：展开浏览器窗口列表，勾选需要导入的标签页
    - 支持多窗口、多标签页同时选择
  </Step>
  <Step title="选择 Notebook 并导入">
    在侧面板选择目标 Notebook，点击「导入到 NotebookLM」。
  </Step>
</Steps>

## 快捷方式

在任意网页上右键 →「导入到 NotebookLM」，无需打开侧面板。

## 排除规则

扩展会自动排除有专用导入工具的平台，避免误操作：

- Bilibili、YouTube、Apple Podcasts、小宇宙 FM
- ChatGPT、Claude、Gemini
- NotebookLM 本身

## 注意事项

- 导入前请确保已打开一个 NotebookLM **笔记本页面**（不是首页）
- 标签页列表仅显示 `http(s)://` 开头的页面
