## 使用方式

NoteFlow 会自动检测当前标签页类型，切换到对应的导入工具。如果当前是普通网页，会显示网页导入界面。

### 单个导入

<Steps>
  <Step title="打开 NoteFlow 侧面板">
    确保已打开一个 NotebookLM 笔记本，然后打开 NoteFlow 侧面板。
  </Step>
  <Step title="输入 URL">
    在输入框中粘贴网页链接，或点击「刷新」按钮直接导入当前页面。
  </Step>
  <Step title="选择 Notebook 并导入">
    在侧面板顶部选择目标 Notebook，点击「导入到 NotebookLM」。
  </Step>
</Steps>

<Tip>
  **更快捷**：在任意网页右键 →「导入到 NotebookLM」，一步完成。
</Tip>

### 批量导入

一次导入多个 URL。

1. 在网页导入界面中输入多个 URL，每行一个
2. 点击「批量导入」
3. 扩展逐个导入，进度条实时显示

<Note>
  也可以点击「导入所有打开的标签页」，自动获取当前所有标签页的 URL 并填入列表。
</Note>

### 右键菜单

浏览任意网页时：
- **右键页面空白处** →「导入到 NotebookLM」→ 导入当前页面
- **右键链接** →「导入到 NotebookLM」→ 导入链接目标页面

## 特殊平台支持

部分平台需要通过浏览器标签页渲染才能提取内容，NoteFlow 会自动处理：

- **X.com (Twitter) 长文** — 自动识别 Article 格式，通过标签页渲染提取完整内容
- **微信公众号** — 绕过反爬机制，浏览器内渲染后提取（详见 [微信公众号导入](/guides/wechat)）
- **Substack** — 专门噪音过滤，精准提取正文（详见 [Substack 导入技巧](/guides/substack)）

## 导入失败？

如果某些 URL 导入失败，NoteFlow 会显示失败列表，支持一键重试。

<Warning>
  导入前请确保已打开一个 NotebookLM **笔记本页面**（不是首页）。
</Warning>
