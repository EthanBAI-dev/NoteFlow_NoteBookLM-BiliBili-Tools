## Supported Import Sources

| Type | Sources |
|---|---|
| 🎬 **Bilibili** | Video subtitles, creator uploads, favorites, series/seasons; per-video or merged import, multi-format output (TXT/MD/JSON/SRT) |
| ▶️ **YouTube** | Videos, playlists, channels with pagination support |
| 🎙️ **Podcasts** | Apple Podcasts & Xiaoyuzhou FM — auto-detect page, download audio |
| 🌐 **Web Tabs** | Import the current tab or all open tabs; auto-excludes platforms with dedicated import tools |
| 🤖 **AI Chats** | Claude, ChatGPT, Gemini — auto-extract Q&A pairs, select what to import |

## Key Features

- **Side Panel UI** — Auto-detect the current site and match the right import tool
- **Multi-Account** — Switch NotebookLM Google accounts from the side panel
- **Target Notebook** — Select a notebook, import directly without switching tabs
- **Import History** — Review all records, easily retry
- **Onboarding Tour** — 5-step guide after installation

## Security & Privacy

- ✅ Completely free, no sign-up required
- ✅ 100% client-side — no data uploaded
- ✅ [Open source](https://github.com/EthanBAI-dev/NoteFlow_NoteBookLM-BiliBili-Tools), fully auditable
- ✅ Bilingual UI (English & Chinese)
