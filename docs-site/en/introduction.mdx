# Batch Import Anything into NotebookLM 🚀

NotebookLM is the best AI knowledge tool, but getting content into it is painful.

**NoteFlow fixes that.**

<CardGroup cols={3}>
  <Card title="Batch Import" icon="bolt">

  </Card>
  <Card title="Auto-Detect" icon="magic">

  <Card title="Smart Rescue" icon="wrench">

  </Card>

## Supported Content

| Type | Sources |
|---|---|
| 🎬 Bilibili | Video subtitles, creator uploads, favorites, danmaku/comments |
| 🌐 Web Pages | Any URL, Substack, WeChat articles, X.com long-form |
| 📚 Doc Sites | Docusaurus, VitePress, MkDocs, GitBook, Mintlify, and 14+ frameworks |
| ▶️ YouTube | Videos, playlists, channels (with pagination support) |
| 🎙️ Podcasts | Apple Podcasts, Xiaoyuzhou FM |
| 🤖 AI Chats | Claude, ChatGPT, Gemini — auto-extract + share cards |
| 📡 RSS | Any RSS/Atom feed |

## Key Features

- **Side Panel UI** — Auto-detects the current site and matches the right import tool
- **Multi-Account** — Switch between Google accounts for NotebookLM
- **Target Notebook** — Select a target notebook and import directly
- **Share Cards** — Export AI conversations as beautiful images (JPEG/PNG/PDF)
- **Import History** — Review all import records, easily retry
- **Onboarding Tour** — 5-step guide after installation, covers all features
