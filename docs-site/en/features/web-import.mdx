title: 'Web Import'

---

## Feature

NoteFlow automatically detects the current tab type. For regular web pages, it shows the web import interface.

<Steps>
  <Step title="Open the NoteFlow side panel">
    Open the side panel. The extension automatically identifies the current tab.
  </Step>
  <Step title="View page info">
    The side panel shows the current page's title and URL for confirmation.
  </Step>
  <Step title="Select tabs">
    - **Current tab**: selected by default, import directly
    - **All tabs**: expand the browser window list, check the tabs you want to import
    - Multi-window and multi-tab selection supported
  </Step>
  <Step title="Select Notebook and import">
    Choose your target notebook in the side panel and click "Import to NotebookLM".
  </Step>
</Steps>

## Shortcut

Right-click on any web page → "Import to NotebookLM" — no need to open the side panel.

## Exclusion Rules

The extension automatically excludes platforms that have dedicated import tools:

- Bilibili, YouTube, Apple Podcasts, Xiaoyuzhou FM
- ChatGPT, Claude, Gemini
- NotebookLM itself

## Notes

- Make sure you have a NotebookLM **notebook page** open before importing (not the homepage)
- The tab list only shows pages with `http(s)://` URLs
