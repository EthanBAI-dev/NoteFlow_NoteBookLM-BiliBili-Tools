## 支持的导入来源

| 类型 | 来源 |
|---|---|
| 🎬 **Bilibili** | 视频字幕提取、UP 主投稿、收藏夹、系列/合集；支持逐视频或合并导入、多种格式输出（TXT/MD/JSON/SRT） |
| ▶️ **YouTube** | 单个视频、播放列表、频道批量导入，支持分页加载 |
| 🎙️ **播客** | Apple Podcasts 与小宇宙 FM — 自动检测页面，下载音频到本地后导入 |
| 🌐 **网页标签** | 导入当前标签页或所有打开的标签页，自动排除有专用导入工具的平台 |
| 🤖 **AI 对话** | Claude、ChatGPT、Gemini — 自动提取问答对，选择需要的部分导入 |

## 核心亮点

- **侧面板 UI** — 打开即检测当前站点，匹配对应导入工具
- **Google 多账号切换** — 侧面板顶部切换 NotebookLM 的 Google 账号
- **目标 Notebook 选择** — 选择目标笔记本，内容直接导入无需切换页面
- **导入历史** — 查看所有导入记录，轻松重试
- **新手引导** — 安装后 5 步骤引导，快速上手
