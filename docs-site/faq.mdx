---
title: '常见问题'
description: 'NoteFlow 常见问题解答'
---

<AccordionGroup>
  <Accordion title="为什么导入后 NotebookLM 显示为空">
    最常见的原因是**选错了 Notebook**。确保在 NoteFlow 侧面板顶部的 Notebook 选择器中选中了正确的目标笔记本。
  </Accordion>

  <Accordion title="Bilibili 视频没有字幕怎么办">
    NoteFlow 依赖于 Bilibili 的 CC 字幕（AI 自动生成或 UP 主上传）。如果视频没有字幕，扩展会显示提示。可以尝试选择有字幕的视频，或者使用 AI 对话功能自行提取。
  </Accordion>

  <Accordion title="支持导入需要登录的页面吗">
    如果你已经在浏览器中登录了该网站，NoteFlow 可以提取你能看到的内容。但无法绕过付费墙或权限限制。
  </Accordion>

  <Accordion title="如何切换语言">
    点击侧面板右上角的 **中** 或 **EN** 按钮即可切换。扩展也会根据浏览器语言自动选择。
  </Accordion>

  <Accordion title="导入历史存在哪里">
    存储在 Chrome 的本地存储中（`chrome.storage.local`）。可在侧面板的设置中查看和清除。
  </Accordion>

  <Accordion title="播客下载速度很慢">
    播客音频文件通常较大（50-100MB），下载速度取决于你的网络和音频托管方的带宽。扩展本身不限速。
  </Accordion>

  <Accordion title="侧面板没有自动检测当前页面">
    如果页面路由变化后侧面板没有刷新，可以点击侧面板顶部的刷新按钮（🔄）手动触发重新检测。
  </Accordion>

  <Accordion title="如何卸载扩展">
    Chrome → 右键扩展图标 → 「从 Chrome 中移除」。或者进入 `chrome://extensions` 找到 NoteFlow 点击「移除」。
  </Accordion>
</AccordionGroup>
